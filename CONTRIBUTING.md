# CONTRIBUTING

이 프로젝트는 DKIS 수학 멘토링 활동을 이어가기 위한 오픈소스형 학습 도구 모음입니다.

## 기여할 수 있는 방법

1. 새로운 학생 데이터를 익명화하여 `data/` 형식에 맞게 추가
2. 새로운 오류 유형 사례를 `docs/student_case_examples.md`에 추가
3. 추천 규칙을 개선
4. 다른 수학 단원의 Manim 시각화 코드 추가
5. README와 실행 가이드 보완

## 데이터 추가 규칙

- 학생 이름, 학번, 연락처 등 개인정보는 절대 포함하지 않습니다.
- 학생 ID는 `S01`, `S02`처럼 익명 ID를 사용합니다.
- 새로운 데이터는 기존 CSV 열 이름을 유지합니다.

## 코드 수정 규칙

- Colab 노트북은 실행 순서가 위에서 아래로 자연스럽게 이어지도록 유지합니다.
- Manim 코드는 장면 목적이 주석으로 드러나도록 작성합니다.
- 큰 모델을 추가할 경우, 보고서에서는 과장하지 않고 파일럿 실험으로 설명합니다.
