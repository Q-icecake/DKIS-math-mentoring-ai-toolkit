# 개인정보 보호 안내

이 저장소는 공개 가능한 오픈소스형 자료로 정리되었지만, 학생 개인정보 보호를 최우선으로 합니다.

## 공개하지 않는 자료

- 학생 이름
- 학번
- 연락처
- 원본 설문 응답 중 개인을 식별할 수 있는 내용
- 실제 성적 또는 평가 결과

## 공개 가능한 자료

- 익명화된 샘플 데이터
- 오류 유형 분류 기준
- 모델 실험 코드
- Manim 시각화 코드
- 교육 자료 검토 문서

## 데이터 사용 원칙

1. 학생 ID는 `S01`, `S02`와 같은 익명 ID를 사용합니다.
2. 공개용 데이터는 실제 학생 원자료가 아니라 익명화된 샘플 형태로 정리합니다.
3. 모델 결과는 학생 평가가 아니라 멘토링 보조 자료로만 사용합니다.
4. 새로운 데이터를 추가할 때도 개인정보를 제거한 뒤 업로드합니다.
