# 교사용 운영 매뉴얼

## 목적

본 프로젝트는 학생 멘토링 과정에서 수집한 학습 어려움을 데이터로 정리하고, 학생별 오류 유형에 맞는 학습 자료를 추천하기 위한 파일럿 구조입니다.

## 운영 절차

1. 수요 조사 실시
2. 학생별 취약 단원 및 오류 유형 정리
3. `student_profile_sample.csv` 형식에 맞게 데이터 입력
4. Colab 노트북 실행
5. 추천 결과 확인
6. 멘토가 추천 결과를 교육적으로 검토
7. 필요한 경우 Manim 시각화 자료 활용
8. 활동 후 피드백 기록

## 교사 확인 포인트

- 오류 유형 분류가 학생의 실제 풀이 과정과 맞는지 확인
- 추천 자료가 학생 수준에 비해 너무 어렵지 않은지 확인
- 모델 결과를 학생 평가나 성적 판단에 사용하지 않도록 지도
- 실제 학생 개인정보가 공개 자료에 포함되지 않도록 확인

## 보고서 작성 시 권장 표현

- “완성된 AI 진단 시스템”이 아니라 “파일럿 실험”으로 표현
- “AI가 학생을 가르쳤다”가 아니라 “모델 결과를 멘토링 보조 자료로 활용하였다”로 표현
- “LSTM을 활용해 오류 흐름 데이터를 입력할 수 있는 구조를 확인하였다”로 표현
