# 공공데이터 분석 및 활용

## 1. 국가수준 학업성취도 평가(KICE) 데이터 분석

### 분석 목적

본 프로젝트에서 실시한 설문조사 결과가 특정 집단에 국한된 현상인지 확인하고, 전국 학생들의 수학 학습 특성과 비교하기 위해 국가수준 학업성취도 평가 자료를 분석하였다. 이를 통해 학생들이 실제로 어떤 영역에서 어려움을 겪고 있는지 확인하고 프로젝트의 필요성과 교육적 타당성을 검증하고자 하였다.

---

### 영역별 취약 영역 비교 분석 (설문 vs. KICE)

| 취약 영역 | 설문 응답 (Q3. 다시 정리하고 싶은 단원) | KICE 학업성취도 평가 결과 | 분석 및 시사점 |
|------------|------------------------------------------|----------------------------|----------------|
| 함수 | 7명 | 정답률 47.18% (가장 낮음) | 설문과 국가 데이터 모두 함수 영역을 학생들이 가장 어려워하는 영역으로 지목함 |
| 방정식·부등식 (문자와 식) | 4명 | 실생활 활용 문제 및 부등식 성질 문제에서 약 50% 수준 정답률 | 기계적 계산은 가능하지만 응용 및 개념 이해가 요구되는 문제에서 어려움을 보임 |

---

### 인지적 오류 유형 분석

| 오류 유형 | 설문 데이터 (Q5. 가장 어려운 부분) | 국내 연구 및 KICE 분석 기준 | 해석 |
|-----------|------------------------------------|----------------------------|------|
| Strategy Error (전략 선택 오류) | "어떤 공식이나 방법을 써야 하는지 떠오르지 않음" (9명) | Transformation Error, Process Skill Error | 문제 해결을 위해 어떤 개념과 연산을 적용해야 하는지 결정하지 못하는 상태. KICE에서 서답형·실생활 활용 문제의 정답률이 낮은 원인과 유사 |
| Process Error (문제 접근 오류) | "문제를 보면 무엇부터 생각해야 할지 막막함" (6명) | Comprehension Error, Transformation Error | 문제 상황을 수학적 구조로 변환하는 초기 단계에서 어려움을 겪는 상태 |

---

### 분석 결과

설문조사 결과와 국가수준 학업성취도 평가 자료를 비교한 결과, 학생들은 공통적으로 함수 단원과 문제 해결 과정에서 높은 수준의 어려움을 경험하고 있는 것으로 나타났다.

특히 학생들은 단순 계산 능력 부족보다는 문제를 이해하고 해결 전략을 수립하는 과정에서 더 큰 학습 부담을 느끼고 있었다.

### 해석

설문조사 결과와 국가수준 학업성취도 평가 자료를 종합한 결과, 학생들의 학습 부진은 단순 계산 능력의 부족보다 문제를 이해하고 해결 전략을 수립하는 과정에서 발생하는 경우가 많음을 확인하였다.

이에 따라 본 프로젝트에서는 학생의 오류를 세분화하여 분석하고, 개인별 학습 특성에 맞는 맞춤형 학습 지원 방안을 설계하고자 하였다.

# 2. AI Hub 수학 교과 문제 풀이과정 데이터 분석

## AI Hub 데이터 개요

AI Hub의 「수학 교과 문제 풀이과정 데이터」는 문제, 풀이 과정, 정답, 해설 정보를 포함하고 있으며 학생의 사고 과정을 단계별로 기록하고 있다는 특징을 가진다.

이를 통해 학생이 어느 단계에서 오류를 발생시키는지 분석할 수 있으며, 최근 교육 분야의 Learning Analytics(학습 분석) 연구에서도 활용되고 있는 대표적인 교육 데이터셋이다.

---

## 분석 목적

학생들의 오답을 단순히 정답 여부로 판단하는 데 그치지 않고, 풀이 과정에서 발생하는 오류를 체계적으로 분석하기 위한 방법론을 검토하고자 AI Hub의 「수학 교과 문제 풀이과정 데이터」를 활용하였다.

본 프로젝트에서는 해당 데이터를 직접 분석하기보다는 학생들의 오류를 구조적으로 분류하기 위한 참고 자료로 활용하였다.

---

## 오류 유형 분류 체계 구축

AI Hub 데이터의 풀이 단계 구조를 참고하여 학생들의 오류를 다음과 같이 분류하였다.

| 오류 유형 | 설명 |
|-----------|------|
| Macro Error | 개념 이해 부족 |
| Micro Error | 계산 및 부호 실수 |
| Process Error | 문제 접근 실패 |
| Strategy Error | 풀이 전략 선택 실패 |

특히 설문조사 결과에서 문제 접근 과정과 풀이 전략 선택에 어려움을 느낀다는 응답이 많았기 때문에, Process Error와 Strategy Error가 학생들의 학습 부진에 중요한 영향을 미치는 요인임을 확인할 수 있었다.

---

## 해석

AI Hub 수학 교과 문제 풀이과정 데이터는 학생의 사고 과정과 오류 유형을 분석하는 방법론을 검토하기 위한 참고 자료로 활용하였다.

이를 통해 학생의 정답 여부뿐 아니라 문제 해결 과정 전체를 분석하는 접근 방식의 중요성을 확인할 수 있었다.
