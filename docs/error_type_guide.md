# Error Type Guide

## 1. Macro Error (개념 이해 오류)

의미
- 수학 개념 자체를 이해하지 못한 경우

특징
- 정의나 원리를 설명하지 못함
- 공식의 의미를 모름
- 개념 간 관계를 이해하지 못함

예시
- 함수의 의미를 설명하지 못함
- 기울기의 의미를 모름
- 확률의 개념을 이해하지 못함

---

## 2. Micro Error (계산 오류)

의미
- 개념은 알고 있으나 계산 과정에서 실수하는 경우

특징
- 부호 실수
- 계산 실수
- 숫자 오기

예시
- 음수 계산 실수
- 식 전개 오류
- 숫자를 잘못 옮겨 적음

---

## 3. Process Error (문제 접근 오류)

의미
- 문제를 어떻게 시작해야 할지 모르는 경우

특징
- 접근 방법이 떠오르지 않음
- 문제의 조건을 해석하기 어려움
- 풀이 순서를 결정하지 못함

예시
- 문제를 보면 무엇부터 해야 할지 모르겠음
- 조건이 많으면 정리가 안 됨
- 어떤 정보를 먼저 사용해야 하는지 모름

---

## 4. Strategy Error (전략 선택 오류)

의미
- 어떤 공식이나 방법을 사용해야 할지 모르는 경우

특징
- 적절한 공식을 선택하지 못함
- 문제 유형에 맞는 풀이 전략을 찾지 못함

예시
- 어떤 공식을 써야 할지 모르겠음
- 비슷한 문제는 풀지만 변형 문제는 어려움
- 여러 방법 중 어떤 것을 선택해야 할지 모름
