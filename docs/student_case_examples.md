# Student Case Examples

## 사례 1

학생 응답
"어떤 공식이나 방법을 써야 하는지 떠오르지 않음"

분석
Strategy Error

해석
문제 해결에 필요한 수학적 개념은 알고 있지만, 적절한 공식을 선택하는 과정에서 어려움을 겪고 있다.

## 사례 6

학생 응답  
"개념은 알겠는데 계산 과정에서 자꾸 부호를 틀림"

분석  
Micro Error

해석  
개념 이해 자체는 어느 정도 되어 있지만, 계산 과정에서 부호 처리나 식 정리 실수가 반복되는 상태이다.
