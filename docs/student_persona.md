# Student Persona

## 유형 A : 함수와 문제 접근에 어려움을 느끼는 학생

특징

* 함수 단원을 가장 어려워함
* 그래프와 식의 관계를 이해하기 어려워함
* 문제를 보면 무엇부터 시작해야 할지 막막함
* Process Error가 주로 나타남

추천 학습 방법

* 함수 그래프 시각화 활동
* 그래프와 함수식 연결 연습
* 문제 접근 순서를 단계별로 제시하는 학습 자료 활용

---

## 유형 B : 공식 선택에 어려움을 느끼는 학생

특징

* 개념은 어느 정도 이해하고 있음
* 어떤 공식이나 풀이 방법을 사용해야 할지 떠오르지 않음
* 응용 문제에서 자주 막힘
* Strategy Error가 주로 나타남

추천 학습 방법

* 문제 유형별 풀이 전략 정리
* 사고 과정 매핑 활동
* 풀이 전략 카드 활용

---

## 유형 C : 계산 과정에서 실수가 많은 학생

특징

* 개념 이해는 비교적 양호함
* 계산 과정에서 실수가 자주 발생함
* 검산 습관이 부족함
* Micro Error가 주로 나타남

추천 학습 방법

* 계산 체크리스트 활용
* 단계별 검산 훈련
* 반복 연습 문제 풀이
