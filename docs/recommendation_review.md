# recommendation_review

## 추천 결과 검토

### 사례 1

#### 학생 특성

* 함수 단원을 어려워함
* 문제를 어떻게 시작해야 할지 모름
* 그래프 해석에 어려움을 느낌

#### 모델 추천

* 함수 그래프 시각화 자료
* 풀이 순서 카드

#### 검토

✅ 적절함

함수식과 그래프의 관계를 시각적으로 이해할 수 있도록 지원하며, 문제 접근 순서를 단계별로 안내할 수 있어 학생의 Process Error를 해결하는 데 도움이 될 것으로 판단된다.

---

### 사례 2

#### 학생 특성

* 어떤 공식이나 방법을 사용해야 할지 모름
* 개념은 이해하고 있음
* 응용 문제에서 자주 막힘

#### 모델 추천

* 풀이 전략 가이드
* 문제 유형 분류 자료

#### 검토

✅ 적절함

학생은 개념 부족보다 문제 해결 전략 선택에 어려움을 보이고 있으므로 Strategy Error를 지원하는 자료가 효과적일 것으로 판단된다.

---

### 사례 3

#### 학생 특성

* 계산 실수가 반복됨
* 개념 이해는 양호함
* 검산 습관이 부족함

#### 모델 추천

* 계산 체크리스트
* 단계별 연습 문제

#### 검토

✅ 적절함

학생의 주요 문제는 개념 부족이 아니라 반복적인 계산 오류이므로 개념 설명보다 계산 과정을 점검할 수 있는 자료가 더 효과적일 것으로 판단된다.

---

### 사례 4

#### 학생 특성

* 개념은 배웠지만 정확하게 설명하지 못함
* 공식을 암기 위주로 학습함
* 새로운 문제에 개념을 적용하는 데 어려움을 느낌

#### 모델 추천

* 쉬운 개념 설명 자료
* 개념-문제 연결 활동
* 실생활 예시 중심 자료

#### 검토

✅ 적절함

학생은 Macro Error가 주로 나타나는 유형으로, 공식 암기보다 개념의 의미를 이해할 수 있는 학습 자료가 필요하다. 따라서 개념 중심 자료를 제공하는 것이 적절하다고 판단된다.

---

## 종합 의견

학생들의 어려움은 크게 Macro Error, Micro Error, Process Error, Strategy Error로 구분할 수 있었다. 각 오류 유형에 맞추어 학습 자료를 추천하는 방식은 학생들의 개별적인 학습 요구를 반영할 수 있다는 점에서 교육적으로 타당하다고 판단된다.

특히 설문 결과에서 많이 나타난 Process Error와 Strategy Error에 대한 지원 자료는 학생들의 문제 해결 능력 향상에 도움이 될 것으로 기대된다. 향후 실제 멘토링 데이터를 추가로 반영한다면 추천의 정확도와 활용성을 더욱 높일 수 있을 것이다.
