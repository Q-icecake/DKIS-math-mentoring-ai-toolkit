# Educational Interpretation

설문 결과를 분석한 결과, 학생들은 단순 계산 실수보다 문제 접근 과정(Process Error)과 풀이 전략 선택 과정(Strategy Error)에서 더 큰 어려움을 겪는 것으로 나타났다.

특히 "어떤 공식이나 방법을 사용해야 할지 모르겠다"는 응답과 "문제를 보면 무엇부터 시작해야 할지 모르겠다"는 응답이 높은 비율을 차지하였다. 이는 학생들이 개념을 암기하는 수준은 가능하지만, 실제 문제 상황에서 적절한 전략을 선택하고 적용하는 과정에서는 어려움을 겪고 있음을 의미한다.

또한 설문 결과에서 함수 단원이 어렵다는 응답이 높게 나타났으며, 학생들은 함수식과 그래프의 관계를 직관적으로 이해하는 데 어려움을 보이는 것으로 나타났다. 특히 그래프 해석이나 문제 상황을 함수로 표현하는 과정에서 어려움을 경험하는 학생들이 많았다.

이러한 결과는 학생들에게 단순한 개념 설명보다 문제 해결 과정 자체를 연습할 수 있는 학습 기회가 필요함을 시사한다. 따라서 본 프로젝트에서는 학생의 오류 유형(Macro, Micro, Process, Strategy)을 분석하고, 각 유형에 적합한 맞춤형 학습 자료와 활동을 연결하는 방향으로 멘토링 프로그램을 설계하고자 한다.
